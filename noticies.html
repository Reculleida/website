Ignaguració Web Reculleida Juliol 2017

Ignagurem la web de Reculleida, la web anirà oferint serveis durant aquest any i principios de l'any vinent, per finalment tindre una eina actual i ràpida tant per a usuaris com per als propis socis.

RecuLleida comunicació

info@reculleida.org
07 - 07 - 2017

Un nou canal de comunicació amb l'asociació, el correu electrònic general de Reculleida, un cop rebut el correu es pasarà al departament correxponent i es tramitarà el més ràpid possible.

Amb aquest nou mètode de comunicació es vol apropar l'asociació als socis i al pùblic en general, el correu electrònic aporta rapidesa i seguretat en les comunicacions.
